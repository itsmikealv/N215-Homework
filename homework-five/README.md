# Home

A web project with SCSS styling and live server capabilities.

## Installation

To install all dependencies, run:

```bash
npm install
```

This will install the required development dependencies, including Sass for compiling stylesheets.

## Available Commands

Once dependencies are installed, you can use the following npm scripts:

### Compile SASS
Compiles SCSS files to CSS with watch mode:
```bash
npm run compile:sass
```

This watches the `scss/` directory and automatically compiles `scss/styles.scss` to `css/styles.css` whenever changes are made.

### Start Development Server
Launches a live server for development:
```bash
npm run serve
```

This starts a local development server and automatically refreshes the page when you make changes.
